# AllTheTools - Minecraft NeoForge 1.21.1

AllTheTools è una mod completa per Minecraft NeoForge 1.21.1 che aggiunge una vasta gamma di nuovi strumenti, armature e scudi realizzati con materiali esistenti e nuovi. Ogni materiale ha le proprie statistiche bilanciate e alcuni offrono abilità speciali uniche quando indossati come set completo.

## Funzionalità Principali

### Materiali Aggiunti
La mod introduce equipaggiamento per i seguenti materiali, ordinati per potenza:
`Legno < Flint < Pietra < Deepslate < Ossidiana < Cuoio < Redstone < Lapis < Oro < Rame < Quarzo < Maglia < Carbon Fiber < Ferro < Ametista < Smeraldo < Diamante < Echo < Netherite`

*   **Legno**: Varianti per tutti i tipi di legno (Quercia, Abete, Betulla, Giungla, Acacia, Rovere Scuro, Mangrovia, Ciliegio, Bamboo, Cremisi, Deformato).
*   **Carbon Fiber**: Ottenuto cucinando il carbone in una fornace.
*   **Echo**: Equipaggiamento di fine gioco potenziato con Echo Shards tramite il tavolo da forgiatura (richiede un template speciale trovato nelle Ancient Cities).

### Abilità Speciali
*   **Spada di Ferro**: Brucia le entità che indossano almeno un pezzo di armatura di Flint.
*   **Armatura Redstone**: Emette un segnale luminoso (effetto Glowing) alle entità vicine.
*   **Armatura Lapis**: Riduce la probabilità di rottura dell'incudine e dimezza idealmente i costi di XP (set completo).
*   **Armatura Smeraldo**: Riduce i prezzi degli scambi con i villager (set completo).
*   **Armatura Ossidiana**: Fornisce l'effetto Fire Protection I integrato.
*   **Armatura Echo**: Rende il giocatore impercettibile a Warden e sensori di sculk (set completo).
*   **Strumenti Echo**: Non emettono rumore quando impugnati (anche nella mano secondaria).

### Compatibilità e Integrazioni
*   **Backported Spears**: Aggiunge lance per tutti i materiali della mod, compatibili con gli incantesimi delle lance.
*   **Scudi Personalizzati**: Aggiunge scudi per ogni variante di legno.
*   **Strumenti in Cuoio**: Strumenti in cuoio colorabili (come le armature).

## Configurazione
La mod include un file di configurazione (`allthetools-common.toml`) che permette di:
*   Abilitare/Disabilitare ogni tipo di materiale.
*   Attivare/Disattivare le abilità speciali delle armature e degli strumenti.

## Installazione
1.  Assicurati di avere installato **NeoForge 1.21.1**.
2.  Scarica la mod e inseriscila nella cartella `mods` del tuo profilo Minecraft.
3.  (Opzionale) Installa **Backported Spears** per sbloccare le lance personalizzate.
